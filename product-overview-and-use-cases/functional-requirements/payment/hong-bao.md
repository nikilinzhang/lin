获取渠道：

```
1.递交租房EOI获取10刀专红。
2.递交表单申请大师兄大师姐通过后台审核后得10刀专红。
3.递交表单申请校园大使通过后台审核后得10刀专红。
4.递交避险救助表单通过后台审核后得10刀专红。
5.皇包车扫码提交照片后得5刀专红。
6.在发现中提交申请宝主通过后台审核后得25刀专红。
7.在发现中提交申请为别人打赏后得5刀专红。
8.在发现中被别人打赏得专红。
9.充值得专红。（冲多少送所冲金额的20%，单笔充值送专红上限100。）
10.首页参与红包抽奖。
11.使用余额在熊猫饿了商户消费或使用余额购买管家卡时获取红包。
```

使用渠道：

```
1.在熊猫饿了商家进行消费。
2.在管家中购买“管家卡”（管家卡后台设置为澳币支付时）。
3.在发现中为宝主打赏。
4.在设置了“1.交易金额为多少以上就不可以使用方式三。2.专红在单笔交易中使用比例。”的商家消费。消费形式为余额➕专红。消费规则根据后台
设置的两个值执行。
5.房客可使用专红抵一部分房租。
6.可用专红租家具。
```
